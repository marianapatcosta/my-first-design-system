# ToggleSwitch
Simple ToggleSwitch component, which mimics the checkbox behavior but with different styling

## Usage
Use ToggleSwitch when the value alternates between two values

- Default ToggleSwitch:
  <Story name='Default'>
    <ToggleSwitch checked={false}
      leftLabel='off' 
      rightLabel='on' 
    />
  </Story>

- ToggleSwitch With Label:
  <Story name='WithLabel'>
    <ToggleSwitch checked={false}
      leftLabel='off' 
      rightLabel='on' 
      label='ToggleSwitch' 
    />
  </Story>
