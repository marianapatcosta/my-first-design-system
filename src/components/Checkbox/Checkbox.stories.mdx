# Checkbox
Simple checkbox component

## Usage
Use checkbox when the value alternates between true and false

- Default Checkbox:
  <Story name='Default'>
    <Checkbox checked={false} label='Checkbox' />
  </Story>

- Checked Checkbox:
  <Story name='Checked'>
    <Checkbox checked={true} label='Checkbox' />
  </Story>

- Disabled Checkbox:
  <Story name='Disabled'>
    <Checkbox disabled checked={true} label='Checkbox' />
  </Story>
