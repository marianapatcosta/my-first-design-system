# Input

Input field

## Usage
use input field when it is supposed that the user provides some written information

- Default Input:
  <Story name='Default'>
    <Input value='hello' />
  </Story>

- Input with Label:
  <Story name='WithIcon'>
    <Input value='hello' label='Input' />
  </Story>

- Input with Icon:
  <Story name='WithIcon'>
    <Input value='hello' icon={Icon} />
  </Story>

- Input with Error:
  <Story name='WithError'>
    <Input value='hello' errorMessage='This is an error!' />
  </Story>
