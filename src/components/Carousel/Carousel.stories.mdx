# Carousel

Simple slider component

## Usage

Use Carousel to display multiple images, either on portrait or landscape orientation, but only one of them is to be shown at a time. To change the image, the user
should click on the lateral arrows. This component also allows to activate automatic mode, so the images are
dynamically changed after a customizable time interval. It may also show previews of the two previous and two next images,
if `hasPreviews` prop is true.
