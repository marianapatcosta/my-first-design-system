# Toast
Simple Toast component

## Usage
Use Toast to give a temporary message to the user. There are 4 toast types available:
alert, info, success and warning

- Info Toast:
  <Story name='Default'>
    <Toast message="info toast" />
  </Story>

- Alert Toast:
  <Story name='Alert'>
    <Toast message="alert toast" type="alert" />
  </Story>

- Warning Toast:
  <Story name='Warning'>
    <Toast message="warning toast" type="warning" />
  </Story>

- Success Toast:
  <Story name='Success'>
    <Toast message="success toast" type="success" />
  </Story>
