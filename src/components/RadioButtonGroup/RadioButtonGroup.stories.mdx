# RadioButtonGroup

RAdio button group components; groups radio button elements

## Usage

Use radio button group component to display a predefine group of options, represented by RadioButton components,
from which the user should choose only one.

- Default RadioButtonGroup:

  <Story name='Default'>
    <RadioButtonGroup />
  </Story>

- Vertical RadioButtonGroup:
  <Story name='Vertical'>
    <RadioButtonGroup
      items={[
        { label: 'option 1', name: 'name 1' },
        { label: 'option 2', name: 'name 2' },
        { label: 'option 3', name: 'name 3' },
      ]}
      orientation={ORIENTATION.VERTICAL}
    />
  </Story>
