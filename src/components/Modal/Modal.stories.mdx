# Modal
<br/><br/> 

### Displays a message or is used to confirm/cancel an action

Use Modal component to advertise the user about some information, error or other
event (default modal) or to ask to confirm some action (Confirmation modal). COnfirmation modal
requires `isConfirmationModal` and `onConfirm` props
