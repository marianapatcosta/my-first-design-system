# Upload
<br/><br/> 

#### Displays a button to upload files

Use Upload component when it is required to upload a file (audio, image, doc, etc). If `hasPreviews` prop is passed, it displays the
`imagesToPreview` or `filesToPreview`, corresponding to the uploaded files
