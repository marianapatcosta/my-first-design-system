# Accordion
<br/><br/> 

### Displays a title and a content when expanded

Use the Accordion when you need to show a title and the content should only be visible on demand,
by clicking on the component. Both title and content can be string or another components/ HTML elements

<Preview withToolbar>
  <Story name="Standard">
    <Accordion
      title="Accordion title"
        content={
          <div>
            <div>Accordion content</div> <img src={Test} style={{ width: "200px" }} />
          </div>
        }
    />
  </Story>
</Preview>
