# Stepper
<br/><br/> 

### Displays the content subdivided in parts displayed in Step component

Use stepper when makes sense to subdivide some content (e.g. a form) in smaller parts, which
should be rendered in Step component. Validations before enable next step can be added, by passing a 
value or a function in `isValid` field of `stepsrMetadata` prop.

<Preview withToolbar>
  <Story name="Standard">
   <Stepper
      stepsMetadata={[
        { content: <p>Step 1 content</p>, isValid: true },
        { content: <p>Step 2 content</p>, isValid: true },
        { content: <p>Step 3 content</p>, isValid: true},
      ]}
      buttonLabel='finish'
      />
  </Story>
</Preview>
