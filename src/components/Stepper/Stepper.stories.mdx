# Stepper

Stepper component

## Usage

Use stepper when makes sense to subdivide some content (e.g. a form) in smaller parts, which
should be rendered in Step component.

- Landscape Stepper:

  <Story name='Landscape'>
    <Stepper
      stepsMetadata={[
        { content: <p>Step 1 content</p> },
        { content: <p>Step 2 content</p> },
        { content: <p>Step 3 content</p> },
      ]}
      onSubmit={() => console.log('submitted')}
    />
  </Story>

- Vertical Stepper:
  <Story name='Vertical'>
    <Stepper
      orientation='vertical'
      stepsMetadata={[
        { content: <p>Step 1 content</p> },
        { content: <p>Step 2 content</p> },
        { content: <p>Step 3 content</p> },
      ]}
      onSubmit={() => console.log('submitted')}
    />
  </Story>
