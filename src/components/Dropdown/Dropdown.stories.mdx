# Dropdown
Simple Dropdown component based on HTML select element

## Usage
Use dropdown when there are more than two options available and the user should pick one of them.
The options can be primitives or objects. optionKey prop indicates which field should be displayed
in dropdown options and selected option when options are of type Object

- Default Dropdown:
  <Story name='Default'>
    <Dropdown 
      placeholder='select an option' 
      options={['option 1', 'option 2', 'option 3']}
      selectedOption={null}
      />
  </Story>

- Dropdown With label:
  <Story name='WithLabel'>
     <Dropdown 
      placeholder='select an option' 
      options={['option 1', 'option 2', 'option 3']}
      selectedOption={null}
      label='Example'
      />
  </Story>
