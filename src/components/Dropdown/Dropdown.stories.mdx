# Dropdown
<br /><br />

### Displays different options only one of them can be selected

Use dropdown when there are more than two options available and the user should pick one of them.
The options can be primitives or objects. optionKey prop indicates which field should be displayed
in dropdown options and selected option when options are of type Object
