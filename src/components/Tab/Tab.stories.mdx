# Tab
Tab component to show a certain content that will be rendered when the tab is active

## Usage
Tab should be used as child of 'Tabs' component and its content is only rendered when the tab is active

- Default Tab:
  <Story name='Default'>
    <Tab label='Tab 1'/>
  </Story>

- Active Tab:
  <Story name='Active'>
    <Tab label='Tab 1' isActive={true}/>
  </Story>
